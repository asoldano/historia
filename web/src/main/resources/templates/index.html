Features

Git Repository Analysis

Analyze any git repository to identify commits that haven't been properly tested.

Visualization

View interactive visualizations of analysis results to identify patterns and areas for improvement.

Asynchronous Processing

Submit analysis requests and track their progress while they run in the background.

How It Works

  1. Submit a Repository
    Enter the URL of a git repository you want to analyze.
  2. Analysis Runs in Background
    The system clones the repository and analyzes commit history to identify untested commits.
  3. View Results
    Once complete, view detailed results and visualizations to understand testing coverage.
  4. Download Reports
    Download CSV reports for further analysis or integration with other tools.
